# Security

## Supported Versions

Security fixes target the latest published release and `main`.

## Reporting

Please report security issues privately through GitHub Security Advisories for `Ameyanagi/yuru`. Do not open a public issue for a vulnerability until a fix is available.

## Installer Scope

The installers are intended to operate in user space. They install the binary under the user profile, optionally edit shell profile files, and optionally create Yuru config files. They should not require `sudo` or administrator privileges.
